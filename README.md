# 🍔 Global Fast-Food Financial Dashboard (ACC102 Track 4)

## Project Overview
This interactive dashboard is designed for financial analysts to evaluate and compare the financial performance of top Quick Service Restaurants (QSRs) including McDonald's (MCD), Yum! Brands (YUM), and Starbucks (SBUX).

**Live Product Link:** (https://jackie-qsr-analytics.streamlit.app/)

## Key Features
- **Interactive Metric Selection:** Users can switch between Profit Margin, Asset Turnover, and ROE via a dropdown menu.
- **Data Visualization:** Automated line charts showing financial trends from 2019 to 2025.
- **Data Integrity:** Specifically handles negative equity issues (e.g., McDonald's) to ensure accurate ROE calculation.

## How to Run Locally
1. Install dependencies: `pip install -r requirements.txt`
2. Run the app: `streamlit run streamlit_app.py`
